# NexaMind-AI-Chatbot 🤖

An advanced AI chatbot built using **Streamlit + NLP + Machine Learning** that understands **Hinglish, Gen-Z slang, and natural language queries**.

---

## 🚀 Features

* 💬 Hinglish + Gen-Z language support
* 🔥 Smart text normalization (short words → full form)
* 🧠 NLP preprocessing (stemming + lemmatization)
* 📊 TF-IDF + Cosine Similarity response system
* 💾 Chat history saving (JSON based)
* 🎨 Custom UI with background support
* 📁 Multiple chat sessions
* ⚡ Fast & lightweight chatbot

---

## 🛠️ Tech Stack

* Python
* Streamlit
* NLTK
* Scikit-learn

## ⚙️ Installation

### Clone the repository

```
git clone https://github.com/Yeman1801/NexaMind-AI-Chatbot.git
cd NexaMind-AI-Chatbot
```

### Install dependencies

```
pip install -r requirements.txt
streamlit
nltk
scikit-learn
emoji
textblob
```

### Run the app

```
streamlit run app.py
```
